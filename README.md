# Javista Coffee — Landing Page

Basit, şık ve animasyonlu kahveci açılış sayfası. Tek dosyada çalışan HTML, harici CSS ve küçük bir JS animasyonu içerir.

Dosyalar
- `index.html` — Ana sayfa.
- `styles.css` — Tüm stiller dışarı çıkarıldı.
- `README.md` — Proje açıklaması.

Nasıl çalıştırılır
1. Proje klasöründe terminal aç.
2. `open index.html` ile tarayıcıda aç.

GitHub'a yüklemek için aşağıdaki talimatları uygulayın (README altındaki adımları izleyin veya otomatikleştirmek isterseniz bana söyleyin).
